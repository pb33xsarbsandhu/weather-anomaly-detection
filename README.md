# 🌦️ Weather Anomaly Detection Dashboard

## 📌 Project Overview

This project detects unusual weather patterns using temperature and rainfall data.
It is built using Python and Streamlit with an interactive dashboard interface.

## 🚀 Features

* Detects temperature anomalies
* Detects rainfall anomalies
* Monthly & seasonal analysis
* Interactive graphs using Plotly
* Filter by date and city

## 🛠️ Technologies Used

* Python
* Streamlit
* Pandas
* Plotly

## ▶️ How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   streamlit run app.py

## 📁 Files Included

* app.py → Main application
* weather.csv → Dataset
* requirements.txt → Dependencies

## 📊 Output

The dashboard shows:

* Temperature trends
* Rainfall trends
* Anomaly detection
* Data table with results

## 💡 Conclusion

This project helps identify unusual weather patterns using basic statistical analysis and visualization techniques.
